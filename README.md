### What is GuineaPig
* Opensource 3D engine was written in python

* Check smiliar project - http://glumpy.github.io/

## Dependency modules.
 - python 3.5.1
 - Anaconda3-2.4.1 ( https://www.continuum.io/downloads )
 - accelerate 2.0 ( anaconda accelerate library )
 - numpy 1.10.4
 - numba 0.23.1
 - pillow (3.1.1)
 - pyqt4
 - pyopengl 3.1
 - pyopengl_accelerate 3.1
 - libpng 1.6.17
 - pygame 1.9.2

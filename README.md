# GuineaPig
GuineaPig is open source 3d engine project write in python.

## Dependency modules.
 - python2.7.6
 - pyqt4
 - pyopengl 3.1
 - pyopengl_accelerate 3.1
 - opengl_context
 - numpy 1.10
 - pyopencl 2015.2
 - pycuda 2015.1
 - pyautogui 0.9.31

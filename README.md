# GuineaPig
GuineaPig is open source 3d engine project write in python.

Blue Print

Dependency modules.
  - python2.7
  - pyqt4
  - pyopengl 3.1
  - pyopengl_accelerate 3.1
  - opengl_context
  - numpy 1.10
  - pyopencl 2015.2
  - pycuda 2015.1

Feature
  - import .blend, .obj, .fbx
  - shader
  - opengl 4
  - cuda, opencl
  - sound
  - qt ui
